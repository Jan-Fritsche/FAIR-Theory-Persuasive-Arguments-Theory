## Individual Contribution 
This directory contains the components for an alternative model of Persuasive Arguments Theory (PAT).

This folder contains all the files for the simulation of the extended Modell. They R-scripts are to be used in the following order:
1.functions
2.simulations
3.analysis_plots
4.analysis_statistics

The resulting analytical plot is located in the folder plots in this directory

Extendend Source Tabels and VAST-Display are included in the subfolder doc
